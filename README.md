# PhoenixFilamentAI

A [PhoenixFilament](https://github.com/franciscpd/phoenix_filament) plugin for AI capabilities.

## Installation

Add `phoenix_filament_ai` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:phoenix_filament_ai, "~> 0.1"}
  ]
end
```
